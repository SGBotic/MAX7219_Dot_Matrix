# 8x8 SPI Dot Matrix Display Module - MAX7219
Sample code for connecting 8x8 Dot Matrix Display with MAX7219 to Arduino using SPI interface

Wiring:</br>
MODULE ----- UNO/NANO ----- MEGA</br>
===============================</br>
VCC --------- +5V ----------+5V</br>
GND --------- GND --------- GND</br>
DIN --------- 11 ---------- 51</br>
CS (LOAD) --- 10 ---------- 10</br>
CLK --------- 13 ---------- 52</br>
